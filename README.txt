== Description ==

This is the [BootstrapThemes](https://thebootstrapthemes.com) Companion plugin. The plugin generates required post types and options for the following themes:

* [Spa theme](https://thebootstrapthemes.com/relax-spa/)
* [Coach theme](https://thebootstrapthemes.com/coach/)
* [Fitness theme](https://thebootstrapthemes.com/fitness-gym/)


== Installation ==

= Uploading in WordPress Dashboard =

1. Go to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select .zip from your computer or Upload .zip to the `/wp-content/plugins/` directory directly.
4. Click 'Install Now'
5. Activate the plugin through the 'Plugins' menu in WordPress.
